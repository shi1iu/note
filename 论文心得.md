### ✅ 一、目标检测（CFAR）准确率评估指标

你可以使用以下标准的分类检测指标：

| 指标                    | 计算方法                                        | 含义                           |
| ----------------------- | ----------------------------------------------- | ------------------------------ |
| **检测率（PD）**        | TP / (TP + FN)                                  | 能识别出的真实目标比例（概率） |
| **虚警率（PFA）**       | FP / (FP + TN)                                  | 将非目标误判为目标的比例       |
| **准确率（Precision）** | TP / (TP + FP)                                  | 判断为目标中有多少是真正的     |
| **F1值**                | 2 × (Precision × Recall) / (Precision + Recall) | 综合评价指标，越高越好         |

- **TP（True Positive）**：正确检测出的目标
- **FP（False Positive）**：将杂波误判为目标（虚警）
- **FN（False Negative）**：漏掉的目标
- **TN（True Negative）**：正确判定为非目标

👉 在 MATLAB 中，如果你知道真实目标位置和检测结果，就可以以一定容差进行比较统计 TP/FP/FN。

由于跟踪是**连续估计问题**，通常使用以下误差指标来评估：

| 指标                     | 计算方法                       | 含义                         |
| ------------------------ | ------------------------------ | ---------------------------- |
| **均方误差（MSE）**      | `mean((est - truth).^2)`       | 越低越好，表示平均偏离程度   |
| **均方根误差（RMSE）**   | `sqrt(mean((est - truth).^2))` | MSE 开方，更直观             |
| **平均绝对误差（MAE）**  | `mean(abs(est - truth))`       | 与RMSE类似，但对离群点不敏感 |
| **轨迹重合度（可视化）** | 画图对比真实轨迹与估计轨迹     | 直观验证滤波稳定性与拟合程度 |

你可以分维度评估（x方向RMSE、y方向RMSE），或者合成欧氏距离误差：

```twmatlab
matlab
rmse = sqrt(mean((x_est - x_true).^2 + (y_est - y_true).^2));
```
